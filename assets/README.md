# Assets

## Folder structure
- sprites/player/    — character sprite sheets (PNG)
- sprites/enemies/   — enemy sprite sheets (PNG)
- backgrounds/       — parallax background layers (PNG)
- tilesets/          — platform and terrain tiles (PNG)
- props/             — torches, mushrooms, decorative items (PNG)
- effects/           — dust, particles, VFX sprite sheets (PNG)
- ui/                — HUD elements (PNG)
- audio/music/       — background music (OGG preferred)
- audio/sfx/         — sound effects (OGG or WAV)

## Adding assets
Drag files directly into the correct folder in Finder.
Godot will auto-detect and import them on next editor focus.
